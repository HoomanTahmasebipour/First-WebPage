Introduction to CSS

Preliminary notes

Styling the document using Cascading Style Sheets (CSS) can be done in three different ways:

  1. In line styling - styling a part of the webpage by writing it directly into the HTML code. NOT RECOMMENDED
  2. Head styling - creating an attribute in the head of the file, which can be refferenced by any elemnet of the code. NOT RECOMMENDED
  3. Style Sheet - creating a seperate CSS file that contains all styling attributes, which is linked in the head. RECOMMENDED

There are many, many style tags you can use in CSS, including:

  • Color tag, the most simple. There are four ways to define the color you want to select:
    1. rgb(_ _,_ _,_ _) - The placeholder inside the brackets is for a number ranging from can 0 - 255 and dictate how concentrated the red, green, or blue is in the color.
    2. #_ _ _ _ _ _a - The placeholders after the pound symbol are for hexadecimal notation, allowing either numbers (from 0-9) or letters (from A - F) to fill in the spots. Each pair of two dictate the concerntration of a different color (red, green, blue).
    3. rgba(_ _,_ _,_ _,_) - The exact same as rgb, but the fourth slot (for a) signifies the transperancy of the text, and the number ranges from 0(min) to 1 (max).
    4. name the color - You can also use the 174 predefined colors offered by CSS itself, but they're too hard to remember so most people just use one of the above three (primarily the second one).
  • Border tag, which requires three properties that must be defined. The style of the border (solid?, dashed?,..), the width of the border (entered in pixels "px"), and its color (entered using the methods mentioned previously).
  • Background tag, which can either be a color or an image. The image option has properties in of itself that enable you to alter the size of the background and if the image reapeats to cover the whole page or is just stretched until it does.
  • line-through tag, while is what is sounds like. Multiple variants, essentialy adds a line through, under, or over a text.
  • And MORE....

How to create and use a stylesheet

The way you select a part of the code to style are called selectors. Here are eight examples:

  • Element - Selects all instances of a particular elemnt. Example below selects every li tag in HTML code.
    • Ex: li {}
  • ID - Selects a particular instance of an element. Example below is selecting the element in th code that the #hello id is hooked to.
    • Ex: #hello {}
  • Class - Selects all elemnts that are categorized by a class in the HTML code. Example below selects all elements that are hooked with the hello class.
    • Ex: .hello {}
  • Star - Selects the entire HTML structure.
    • Ex: * {}
    Descendant - Used to select a specific descendant of a branch. In the example below, all a tags that are inside an li tag are selected.
    • Ex: li a {}
  • Adjacent - Selects an element that is after (sibling) another specfied element. Example below is selecting all ul tags that come after an h4 tag.
    • Ex: ul + h4 {}
  • Attribute - Selects an attribute inside a tag. example bellow selects all href attributes that point to google.com inside an a tag.
    • Ex: a[href:hhtp://www.google.com] {}
  • Nth of Type - Selects the nth element of a particular type. example below selects the third element in any li tag.
    • Ex: li:nth-of-type(3) {}
NOTE:

Inheritance: If a style is applied to a tag that is the parent of other tags, its children will inherit that style.

Specificity: If a style is applied to the parent tag, and then another is applied to something within that tag, the more specific selector's trait is applied and the less specfic one is ignored.
